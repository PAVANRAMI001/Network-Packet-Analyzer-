# 🛠️ Network Packet Analyzer CLI

A simple and efficient **command-line tool** to analyze network packet capture (**PCAP**) files and generate useful insights such as packet count, protocols, and network activity summary.

---

## 📌 Project Purpose

This project was created to simplify the analysis of **PCAP (Packet Capture)** files, commonly generated by tools like **Wireshark**.
Instead of manually inspecting packets, this CLI tool automates the process and provides a quick summary of network traffic.

---

## 🎯 Project Objectives

* Analyze `.pcap` files from network captures
* Extract useful information such as:

  * Total packets
  * Unique IP addresses
  * Protocol distribution
* Provide a simple and user-friendly **CLI interface**
* Generate structured output (JSON / reports)
* Keep the tool lightweight and fast

---

## ⚙️ Features

* 📂 PCAP file analysis
* 📊 Protocol breakdown (TCP, UDP, ICMP, etc.)
* 🌐 Unique IP detection
* ⚡ Fast and efficient processing
* 🖥️ Command-line interface support

## 🧠 How It Works

1. User provides a `.pcap` file
2. Tool reads packet data
3. Extracts:

   * Packet count
   * IP addresses
   * Protocol types
4. Displays analysis in terminal

## ⚠️ Limitations

* Limited deep packet inspection
* No GUI (only CLI-based)
* Works best with standard PCAP formats
* Large files may take more time to process
